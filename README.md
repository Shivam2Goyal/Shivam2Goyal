## About Me

I’m a B.Tech student in AI & Data Science at IIT Jodhpur, interested in building systems at the intersection of machine learning, software, and real-world applications. My work spans training machine learning models, designing data pipelines, and developing full-stack applications — I enjoy taking projects from raw data to working, deployable solutions.

I’m particularly interested in problems where data-driven methods meet practical systems. Whether it’s building an intrusion detection system with end-to-end ML workflows, developing interactive data visualization platforms, or implementing efficient backend systems, I focus on creating solutions that are both technically sound and practically useful.
